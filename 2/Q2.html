Q2. Create a liquid layout with HTML, CSS, and Javascript. This layout must support the following resolutions: 1024x768, 1680x1050. Explain why you would use a liquid layout.

 

Liquid Layout

Liquid Layouts are used for making a webpage veiwable on any size screen, from large computer screens all the way to the tiny screens on phones.

This entire application is built using a liquid layout, stretching the window will cuase the text and boxes to shrink and move around to fit the new size.

With hundreds and thousands of visiters going in and out of your site, you want to make sure that all of the options are avaliable for which ever way the visiter wishes to open the site.
